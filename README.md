# Intern Pilot

Intern Pilot is a full-stack internship and job application tracker built using the MERN stack. It helps students and job seekers manage applications, track progress, analyze resumes against job descriptions using AI, and gain actionable insights to improve their chances of getting selected.

Built as part of the Digital Heroes Internship Assignment.

---

##  Features

### Application Tracker
- Add internship/job applications
- Update application status
- Delete applications
- Track company, role, platform, and application date

### Analytics Dashboard
- Total Applications
- Applied Count
- Online Assessments
- Interviews
- Rejections
- Selections
- Success Rate

### Visual Insights
- Status Distribution Pie Chart
- Platform-wise Application Analysis
- Application Timeline Graph

### AI Resume Analyzer
Upload a PDF resume and compare it against a job description.

The AI provides:

- ATS Match Score
- Resume Summary
- Matching Skills
- Missing Skills
- Resume Improvement Suggestions
- Personalized Learning Roadmap
- Learning Resources

Powered by:
- Groq API
- Llama 3.3 70B Versatile

## Installation

### Clone Repository

```bash

```

### Frontend Setup

```bash

npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
